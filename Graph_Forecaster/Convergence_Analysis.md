## **PART VII: CONVERGENCE SCORE CALCULATION**

### **7.1 Algorithm**

```python
def calculate_convergence_score(year):
    # Step 1: Build dynamic graph
    graph = build_domain_graph(year)
    
    # Step 2: Calculate density
    total_edges = count_cross_domain_edges(graph)
    max_possible = n_domains * (n_domains - 1) / 2
    density = total_edges / max_possible
    
    # Step 3: Calculate intensity
    edge_history = get_edge_history(year - 5, year)
    intensity = derivative(edge_history, order=1)
    acceleration = derivative(edge_history, order=2)
    
    # Step 4: Emergence mentions
    mentions = count_emergence_phrases(year)
    mention_growth = (mentions[year] - mentions[year-2]) / mentions[year-2]
    
    # Step 5: Multi-layer signals
    layers = {
        'L1_science': score_publications(year),
        'L2_patents': score_patents(year),
        'L3_investment': score_funding(year),
        'L4_media': score_cultural_products(year),
        'L5_social': score_social_discourse(year),
        'L6_political': score_regulatory(year),
        'L7_linguistic': score_frame_shift(year),
        'L8_infrastructure': score_technical_readiness(year)
    }
    
    weights = {
        'L1': 0.30, 'L2': 0.15, 'L3': 0.10, 'L4': 0.08,
        'L5': 0.07, 'L6': 0.12, 'L7': 0.10, 'L8': 0.08
    }
    
    multi_layer_score = sum(weights[k] * layers[k] for k in layers)
    
    # Step 6: Combine metrics
    convergence_score = (
        0.35 * normalize(density) +
        0.25 * normalize(intensity) +
        0.20 * normalize(acceleration) +
        0.20 * multi_layer_score
    )
    
    return convergence_score * 100  # Scale to 0-100
```
